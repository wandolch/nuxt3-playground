## Setup

Make sure to install the dependencies:

```bash
# npm
npm install
```

## Running Mocks Server
```bash
npm i -g http-server
cd mocks
http-server ./ --cors
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```

## Production

Build the application for production:

```bash
# npm
npm run build
```

Locally preview production build:

```bash
# npm
npm run preview
```
